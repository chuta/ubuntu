## **4. Product Definitions (from website FAQ)**

### **4.1 Ubuntu Tribe**

An ecosystem designed to democratize access to gold. Not merely a token issuer — a platform connecting real-world asset backing with modern digital access.

### **4.2 $GIFT (Gold International Fungible Token)**

- Digital certificate of gold ownership
- 100% backed by physical gold, ethically sourced
- Each token represents **1 mg of physical gold**
- Accessible from **less than $1**
- Stored in world-class, audited, insured vaults globally
- LBMA-standard gold; custody in Singapore, Switzerland, Dubai
- Proof of reserves via third-party audits + blockchain verification

### **4.3 Utribe Wallet**

Platform for buying, selling, holding, transferring, and managing $GIFT and other supported assets. Mobile-first with Send, Receive, Buy GIFT Gold, and Withdraw actions.

### **4.4 How Ubuntu Tribe Works (4-step flow)**

1. **Real Gold Sourcing** — Physical gold via structured, responsible channels
2. **The Ubuntu Tribe System** — Custody, verification, audits, trust infrastructure
3. **Utribe Wallet Platform** — Mobile-first digital value access
4. **Digital Gold Ownership** — GIFT as everyday-accessible digital certificate

## **7. GrowthOS Brand Application Guidelines**

GrowthOS is an **internal commercial operations platform** — it should feel like part of the Ubuntu Tribe family while optimizing for CRM information density.

| Element | Guideline |
| --- | --- |
| **Logo placement** | Top-left nav; login screen center; PDF report header |
| **Primary UI color** | Purple (`#9035F4`) for nav, primary buttons, active states |
| **Accent color** | Gold (`#C9932A`) for pipeline value, deal highlights, success states |
| **Background** | Light mode default (white/light gray content area); purple header/nav bar |
| **Typography** | Clean sans-serif (match website if identifiable at build time; fallback: Inter or system-ui) |
| **Tone** | Professional, credible, human — not crypto-trader aesthetic |
| **Product naming in UI** | Use official names: $GIFT, Utribe Wallet, UbuntuVerse, Ubuntu Capital |
| **External link** | Footer or About: "Ubuntu Tribe" links to https://utribe.one/ |
